## 📖 About the Game

**Guess Who?** is a bilingual (Arabic/English) multiplayer guessing game inside VRChat.  
Two players face off — each gets a secret image, and they take turns asking yes/no questions to figure out what the other player's image is.  
The first player to correctly guess wins the round!

**Total Players:** 115,206+

---

## 🕹️ How to Play

1. Choose a category (e.g. Anime, Disney, Football Players, Movies...)
2. Join your team to get assigned a random image
3. Take turns asking yes/no questions using the in-game buttons
4. When you're confident, guess directly: *"Are you [name]?"*
5. If they answer **"Yes"** — press the **Win** button to end the round!
